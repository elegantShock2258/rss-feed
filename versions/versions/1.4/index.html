Refreshed Discover Page, Categories in Articles, and UI Improvements

Changelog:

  • Refreshed Discover-page, cleaner user interface and easier navigation
  • New recommended-section in Discover-page
  • Small tweaks to the app colors (devices running Android 12 or newer)
  • Categories section in article reader


Discover Page

The Discover page has undergone a slight UI redesign, with a reduction in the size of categories at the top and simplification of the results below.

Additionally, a new recommended section has been introduced, displaying a list of tested feeds. "Tested" means that the feed currently functions as expected and delivers valid content. Naturally, this status may change over time, and the list will be actively monitored for irregularities.

Categories in Articles

If the post publisher provides categories along with the post, they are now visible when reading the article.

For this feature to function correctly, the feed provider must include proper <category> tags with appropriate details.

Other changes

The app colors are slightly tweaked, especially in light mode.

These color changes target primarily devices running Android 12 or newer with Material You dynamic colors support.



Source list items also got an UI redesign, now the component is the same across Discover- and Sources-tab.
